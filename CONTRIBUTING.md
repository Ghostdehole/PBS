## How to Contribute

### 1. Report Issues
If you find a bug or have a feature suggestion, please open an issue describing:
- Steps to reproduce
- Current behavior vs expected behavior

### 2. Fork & Clone
```bash
git clone https://github.com/Ghostdehole/PBS.git
cd PBS
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature
```

### 4. Code Style
This project uses modern C++ (C++17). Please follow these guidelines:
 - Clear and meaningful naming
 - Proper and concise comments
 - Minimal dependencies

### 5. Commit & Pull Request
```bash
git add .
git commit -m "Brief description of your changes"
git push origin feature/your-feature
```
Then submit a Pull Request on GitHub.
