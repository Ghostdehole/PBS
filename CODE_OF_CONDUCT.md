# Code of Conduct

This project welcomes everyone and is committed to creating an open, inclusive, and respectful community.

## Expected Behavior
- Be respectful of others' opinions and communicate politely
- Focus on issues, not personal attacks
- Use polite language in issues and pull requests

## Unacceptable Behavior
- Discrimination, harassment, or abusive language
- Posting hateful, threatening, or obscene content
- Intentionally disrupting the work of others
